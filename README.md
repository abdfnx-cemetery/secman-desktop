## Build & Run

> First sign up via [**Secman Authorize Website**](https://auth.secman.dev)

### Requirements

- [**Node.js**](https://nodejs.org)
- [**Vue**](https://vuejs.org)

### Run

``` bash
# install deps
yarn

# run secman desktop in dev mode
yarn serve

# build application for production
yarn build
```
